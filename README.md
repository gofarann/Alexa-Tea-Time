Alexa Tea Time allows a user to set a timer based on the type of tea being brewed.

White Tea - 1 minute
Green Tea - 3 minutes
Black Tea - 5 minutes
Tisane/ Herbal - 7 minutes

Alexa Tea Time may be invoked by saying "Alexa, ask Tea Time for Green Tea".
